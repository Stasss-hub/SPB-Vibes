# Архитектура SPB Vibes

## Общая архитектура

Выбрана **монолитная архитектура с модульной организацией кода** (модульный монолит). Технологический стек:

| Компонент | Технология |
|-----------|------------|
| Бэкенд | Python (FastAPI) / Go |
| База данных | PostgreSQL 14+ |
| Кэш | Redis |
| Очереди | RabbitMQ (опционально) |
| Фронтенд (моб.) | Flutter |
| Мониторинг | Prometheus + Grafana |

## Интеграции

- Travelpayouts (Aviasales) – поиск авиабилетов
- Sendy – платёжный агрегатор (Alipay/WeChat Pay)
- 2GIS (перспектива) – карты
- NotiSend (перспектива) – push-уведомления

## Хранилища данных

- PostgreSQL: пользователи, корзины, заказы, платежи, промокоды
- Redis: временные задачи поиска авиабилетов (`flight_search_task`)
