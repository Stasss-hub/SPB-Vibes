# SPB Vibes – техническая документация

**SPB Vibes** – это мобильное приложение для китайских туристов в Санкт-Петербурге, объединяющее сервисы бронирования авиабилетов, отелей, музеев, экскурсий и такси с оплатой через Alipay/WeChat Pay.
